# CafeWhereIGo
* 프로젝트명: 카페어디가?!
* 프로젝트 개요
  - 기존의 카페 정보검색의 불편함을 개선하는 서비스 개발
  - 카페에 대한 정보를 한 눈에 알기 쉽게 표시
  - 카페에 대한 리뷰 게시가능
  - 카페 주차장 위치정보를 카카오맵으로 표시
  - 카페 정보검색 외에도 음료주문 및 단체석 예약가능
  - 카페관련 커뮤니티 게시판 존재(오늘의 카페, 이벤트, Q&A, 카페탐방후기)
